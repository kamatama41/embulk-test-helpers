# embulk-test-helpers
Helper classes for Unit test of [Embulk](http://www.embulk.org) plugins.

## Installation
Your build.gradle
```gradle
repositories {
    maven { url 'http://kamatama41.github.com/embulk-test-helpers/repository' }
}

dependencies {
    testCompile 'com.kamatama41:embulk-test-helpers:1.0.1'
}
```
